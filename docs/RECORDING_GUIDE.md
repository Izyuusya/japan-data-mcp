# デモGIF 録画ガイド

README に埋め込むデモGIFの録画手順。

---

## 準備

### ScreenToGif インストール
https://www.screentogif.com/ からダウンロード（ポータブル版でもOK）

### 録画前の設定
- Claude Desktop で japan-data-mcp が動作することを確認
- フォントサイズを **大きめ** に設定（README上で縮小表示されるため）

---

## 録画する動画: 1本

### `docs/demo-chat.gif` — Claude Desktop での実際の使用

**撮影対象**: Claude Desktop のチャット画面

**シナリオ**:
1. Claude Desktop を開く
2. 入力欄に「札幌市の人口を教えて」と入力
3. 送信 → レスポンスが返ってくるのを待つ
4. **出典フッター（出典名・検証URL・取得日時）が表示されるところまで映す**

**ポイント**:
- 出典フッターが映っていることが最重要（差別化ポイント）
- ウィンドウ幅は **800px 程度** に絞る
- 不要な UI 要素（サイドバー等）は閉じておく

**録画設定（ScreenToGif）**:
- フレームレート: 10 fps（十分滑らか + 軽量）
- 録画後エディタで:
  - 入力前の待機フレームを削除
  - レスポンス表示後に **2秒程度停止フレーム** を追加（読む時間）
  - 最後のフレームも **2秒停止**
- GIF出力時:
  - 色数: 256
  - ループ: あり（無限ループ）
  - **目標サイズ: 5MB以下**（GitHub表示で重くならない）

**保存先**: `docs/demo-chat.gif`
